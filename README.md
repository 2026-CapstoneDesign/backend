# 🚀 AI 기반 아르바이트 교육 및 학습 관리 서비스 (Backend)

## 📌 프로젝트 소개

사장님이 매뉴얼(PDF, 문서 등)을 업로드하면 AI가 자동으로 교육 자료를 생성하고, 직원이 학습 및 퀴즈를 수행할 수 있는 AI 기반 교육 플랫폼입니다.

Backend 서버는 인증, 사용자 관리, 매장 관리, 설문, 퀴즈, 학습 관리, 알림 및 대시보드 기능을 제공합니다.

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Passport.js (Google, Kakao OAuth)
* AWS EC2
* Docker

---

## 🏗️ System Architecture

```text
Frontend (React)
       ↓
Node.js + Express API Server
       ↓
MongoDB
       ↓
FastAPI AI Server
       ↓
OpenAI API
```
